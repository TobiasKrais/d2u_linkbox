# d2u_linkbox
Ein paar Boxen mit Bild, Titel und optionalem Link ganz einfach in Redaxo 5 verwaltet

Dieses Addon bietet ein einfaches Modul zur Ausgabe von Boxen, die lediglich ein Bild und einen Titel beinhalten. Diese Boxen können optional mit einem Link versehen werden, und On-/Offline geschalten werden.

Außerdem lassen sich die Boxen einer oder mehreren Kategorien zuordnen, die dann im Modul ausgewählt werden können.

Im D2U Helper Addon gibt es eine Übersicht, welche Übersetzungen von Linkboxen fehlen oder aktualisiert werden müssen.

## Beispielmodule

- 24-1 D2U Linkbox - Linkboxen mit Überschrift in Bild (BS4, deprecated)
- 24-2 D2U Linkbox - Linkboxen mit Überschrift unter Bild (BS4, deprecated)
- 24-3 D2U Linkbox - Farbboxen mit seitlichem Bild (BS4, deprecated)
- 24-4 D2U Linkbox - Slider (BS4, deprecated)
- 24-5 D2U Linkbox - Linkboxen mit Text neben dem Bild (BS4, deprecated)
- 24-6 D2U Linkbox - Linkboxen mit Text und Hoverbild (BS4, deprecated)
- 24-7 D2U Linkbox - Linkboxen mit Überschrift in Bild (BS5)
- 24-8 D2U Linkbox - Linkboxen mit Überschrift unter Bild (BS5)
- 24-9 D2U Linkbox - Farbboxen mit seitlichem Bild (BS5)
- 24-10 D2U Linkbox - Slider (BS5)
- 24-11 D2U Linkbox - Linkboxen mit Text neben dem Bild (BS5)
- 24-12 D2U Linkbox - Linkboxen mit Text und Hoverbild (BS5)
